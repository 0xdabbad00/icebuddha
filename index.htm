Project status: alpha

IceBuddha is a hex viewer and binary file parser that runs in the browser. It is still in development.

The purpose of this project is to leverage the HTML5 FileReader capability, to do analysis locally (entirely client-side). No bandwidth problems for me and no privacy concerns from you of uploading data to my server, because it doesn't happen. This also makes everything fast: The speed of light is so limitting. :)
