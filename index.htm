Project status: alpha

2012.07.12 - IceBuddha is still in development. It is a hex viewer and binary file parser. The binary file parsing is still in the works. It has only minimal functionality at parsing PE files.

The purpose of this project is to leverage the HTML5 FileReader capability, so you can drag and drop a large file here, and it will be read and parsed entirely through javascript locally on the system without needing to upload it, thereby providing instant feedback.

Read more about IceBuddha in my post on 0xdabbad00.com.
