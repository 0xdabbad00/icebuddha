Alpha project

2012.04.03 - IceBuddha is in it's early stages of development. It is a hex viewer and binary file parser. Much of the functionality is missing.

The entire basis of this project is to leverage the HTML5 FileReader capability, so you can drag and drop a large file here, and it will be read and parsed entirely through javascript locally on the system without needing to upload it, thereby providing instant feedback.
